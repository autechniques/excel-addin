EXCEL-BIT

Để cài đặt, hãy chắc chắn bạn đang sử dụng Excel64-bit. Cách kiểm tra phiên bản bit như danh sách phía dưới:

  • Office 2003:  32-bit (Không hỗ trợ)
  • Office 2007:  32-bit (Không hỗ trợ)
  • Office 2010:  File > Help
  • Office 2013:  File > Account > About Excel
  • Office 2016:  File > Account > About Excel
  • Office 2019:  File > Account > About Excel
  • Office 2021:  File > Account > About Excel
  • MS 365 :  File > Account > About Excel
