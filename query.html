QUERY function

● Truy vấn dữ liệu trên một bảng cố định dựa vào câu lệnh SQL.

 VD1: tìm tất cả các chuỗi con bắt đầu với PERF (phần tô màu).

 VD2: Triển khai bảng thống kê ở dạng PIVOT (truy vấn không hỗ trợ keyword PIVOT).

 VD3: Tính toán số lượng nguyên liệu tồn kho.

=QUERYZDAU(A9:B14,"SELECT NGUYEN_LIEU,SUM(TOTAL) AS SUM FROM (SELECT TEN_SET,NGUYEN_LIEU, DON_VI*SO_LUONG AS Total FROM (SELECT * FROM TB_DINHMUC LEFT JOIN TABLE ON TB_DINHMUC.TEN_SET = TABLE.TEN_SET)) GROUP BY NGUYEN_LIEU")

Chú ý:

 1. Query là hàm truy vấn bảng động (Mặc định nó chỉ thực hiện truy vấn lên bảng Table duy nhất)

 2. Để truy vấn 2 bảng trở lên (Bạn cần đặt tên vùng và sử dụng công cụ XQUERY từ Ribbon để tạo bộ dữ liệu mới)
