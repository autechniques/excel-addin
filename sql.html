SQL function

● Là một trong các hàm chủ lực cùng với nhóm các siêu hàm GROUP, XGROUP, GROUPBY, XGROUPBY, XSQL.
● Sử dụng để truy vấn các bảng dữ liệu cố định trên Excel.
● Thay thế Pivot table và Power Query cho hầu hết các tác vụ báo cáo, thống kê.

 VD1: Phân tách dữ liệu ô (các chuỗi) tại cột C thành nhiều dòng.

 VD2: Tính lũy kế cột D phân nhóm theo 2 tiêu chí B,C.

 VD3: Lọc bỏ các hàng trống C,D,E,F (lọc bỏ nếu tất cả giá trị =0).

 VD4: Pivot bảng dữ liệu.

 VD5: Nhóm các chỉ mục tại cột B theo nhóm cột C.

 VD6: Xoay hàng thành cột với tính năng UNPIVOT.

 VD7: Phân tách cột thành dòng.

 VD8: Thống kê từ và số lần xuất hiện của chúng.

=SQL.AU(A1:A28,"SELECT T.ITEM,COUNT(T.ITEM) AS V FROM TABLE.COLUMNS(A).SPLIT(', ') AS T GROUP BY ITEM",0,0)

 VD9: Thống kê TOP 5 tổng lớn nhất.

 VD10: Lọc danh sách học sinh nữ trong lớp (Đặt N trước chuỗi ký tự Unicode).

=SQL.AU(A2:C30,"SELECT TOP 5 A,B,SUM(C) AS D FROM TABLE GROUP BY A,B ORDER BY D DESC",0,0)

Điều kiện:

 1. Để sử dụng các hàm SQL và XSQL bạn cần cài đặt tối thiếu SQL Server Express (bản miễn phí) hoặc SQL Server Enterprise (bản trả phí).

 2. Nên sử dụng phiên bản SQL Server mới nhất (ví dụ SQL Server 2022 thay vì SQL Server 2019).

 3. Tham khảo thêm các ví dụ khác tại Videos.

Lưu ý:

 1. Tham chiếu [columns], hoặc tham số của các Aggregate function (SUM, MAX, MIN, AVG) nếu là expression phải đặt trong mảng {..}.

 2. Các tham số [order], [pre_num], [next_num] có thể [>0, =0, <0] hoặc missing .
