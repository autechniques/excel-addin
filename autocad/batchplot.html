BATCHPLOT Command

● In hàng loạt model/layout từ một hoặc nhiều file/thư mục.

Giao diện:

Hỗ trợ đa dạng đối tượng khung in + kiểu in + kiểu xuất:

 Tùy chọn các kiểu khung in: block (simple, anonymous, dynamic, xref..), line, pline.

 Hỗ trợ các hướng in (theo tên attribute block hoăc theo hướng sắp xếp block hoặc tự chọn) cũng như hướng xoay bản vẽ..

 Tùy chọn in trên layout hiện hành, nhiều layouts hoặc nhiều files.

 Trường hợp xuất pdf, hỗ trợ merge các bản vẽ trong cùng layouts hoặc cùng thư mục.

 Tên file xuất có thể đặt tùy chỉnh hoặc được đặt tự động theo tên file + vị trí layout.
